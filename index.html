Algorithms

Bubble Sort

Bubble Sort is one of the simplest sorting algorithms

Insertion Sort

Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time.

Merge Sort

Merge Sort is a Divide and Conquer algorithm.

Quick Sort

Quick Sort is one of the most famous and effective Sorting Algorithms. It is also an example of divide and conquer algorithm.

Minimum Spanning Tree

Minimum spanning tree is the spanning tree where the cost is minimum among all the spanning trees.

A* Path Finding

A* is the most popular choice for pathfinding, because it's fairly flexible and can be used in a wide range of contexts
